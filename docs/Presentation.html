💡 Key Design Decisions

Architecture Choices

  • ActorHelper as single entry point — Tests stay clean
  • BaseTest for auth only — Separation of concerns
  • HashMap as data carrier — Decouples JSON from POJOs
  • Static URLGenerator — Centralized endpoint mgmt
  • createPayload() in POJO — Each POJO owns its structure

Quality & Safety

  • Fail-fast (Exception on non-200) — Issues caught immediately
  • prettyPrint() on all responses — Full debugging visibility
  • Config externalized — No hardcoded credentials
  • Quality gates per step — No skip allowed
  • PR only on 100% pass — Enforced quality
